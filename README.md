# The Temple of Python: A Terminal Adventure Game 🎮👾

A text-based role-playing adventure game (RPG) built using core Python structures. **The Temple of Python (v4.0)** features a dynamic narrative engine, runtime inventory tracking, mathematical logic challenges, and an unpredictable, randomized boss battle system—all running within a clean CLI ecosystem.

---

## 🌟 Key Features

- **Text Typewriting Blueprint:** Implements a localized temporal streaming effect (`type_effect`) to mimic retro arcade dialogues.
- **Dynamic Inventory & State Management:** Tracks player health matrices and utility collectibles (e.g., Rusty Sword, Golden Key) in real-time.
- **Logical Branching Decisions:** Implements puzzle-solving corridors, riddles, and active environmental hazards.
- **The Ultimate Bug - Boss Fight:** An advanced battle engine utilizing pseudo-random outcome generators (`random.choice`) to simulate critical hits and dynamic damage tracking.
- **v4.0 Score Optimization:** Features a running scoreboard system with performance-based end-game multipliers (Health Bonuses).

---

## 🛠️ Built With

- **Language:** Python 3.x
- **Core Concepts Used:**
  - Standard User Interaction (`input` & `print`)
  - Procedural Control Blocks (Modular Functions)
  - Deterministic Flow Controls (`while` loops and nested `if-else` blocks)
  - Algorithmic State Modifiers (List manipulation & compound assignment operators)
